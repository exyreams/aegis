# Collateral Management API

The Collateral Management API handles collateral pools, valuations, margin calls, substitutions, and liquidation workflows for secured lending.

## Base Endpoint

```
/api/collateral
```

## Collateral Pools

### List Collateral Pools

Get all collateral pools for the authenticated party.

```http
GET /api/collateral/pools
```

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "contractId": "00abc123...",
      "poolId": "POOL-001",
      "borrower": "Alice",
      "lender": "Bob",
      "loanId": "LOAN-001",
      "assets": [
        {
          "assetType": "BTC",
          "amount": "10.5",
          "currentValue": "1200000.00"
        }
      ],
      "totalValue": "1200000.00",
      "requiredValue": "1000000.00",
      "collateralRatio": "1.20",
```

Retrieve a specific collateral pool.


**Parameters:**

## Margin Calls

### Trigger Margin Call

Lender triggers a margin call when collateral ratio falls below threshold.

```http
POST /api/collateral/pools/:contractId/margin-call
```

**Parameters:**
- `contractId` (path): The collateral pool contract ID

**Request Body:**

```json
{
  "requiredAdditionalCollateral": "50000.00",
  "deadline": "2024-06-10T00:00:00Z"
}
```

**Response:**

```json
{
  "success": true,
  "message": "Margin call triggered",
  "data": {
    "marginCallId": "MC-001",
    "deadline": "2024-06-10T00:00:00Z"
  }
}
```

### Respond to Margin Call

Borrower responds to a margin call by adding collateral.

```http
POST /api/collateral/pools/:contractId/respond-margin-call
```

**Parameters:**
- `contractId` (path): The collateral pool contract ID

**Request Body:**

```json
{
  "additionalAssets": [
    {
      "assetType": "ETH",
      "amount": "25.0",
      "value": "50000.00"
    }
  ]
}
```

**Response:**

```json
{
  "success": true,
  "message": "Margin call satisfied",
  "data": {
    "newTotalValue": "1200000.00",
    "newCollateralRatio": "1.20"
  }
}
```

## Liquidation

### Initiate Liquidation

Lender initiates liquidation when collateral ratio falls below liquidation threshold.

```http
POST /api/collateral/pools/:contractId/initiate-liquidation
```

**Parameters:**
- `contractId` (path): The collateral pool contract ID

**Request Body:**

```json
{
  "reason": "Collateral ratio below liquidation threshold",
  "outstandingDebt": "1000000.00"
}
```

**Response:**

```json
{
  "success": true,
  "message": "Liquidation initiated",
  "data": {
    "liquidationContractId": "00ghi789..."
  }
}
```

### Execute Liquidation

Execute the liquidation of collateral assets.

```http
POST /api/collateral/liquidations/:contractId/execute
```

**Parameters:**
- `contractId` (path): The liquidation contract ID

**Request Body:**

```json
{
  "liquidationPrice": "1050000.00",
  "liquidationCosts": "10000.00"
}
```

**Response:**

```json
{
  "success": true,
  "message": "Liquidation executed",
  "data": {
    "settlementContractId": "00jkl012...",
    "netProceeds": "1040000.00",
    "debtCovered": "1000000.00",
    "surplus": "40000.00"
  }

```

### Cancel Liquidation

Cancel a liquidation if borrower satisfies the debt.

```http
POST /api/collateral/liquidations/:contractId/cancel
```

**Parameters:**
- `contractId` (path): The liquidation contract ID

**Request Body:**

```json
{
  "reason": "Debt satisfied"
}
```

**Response:**

```json
{
  "success": true,
  "message": "Liquidation cancelled"
}
```

## Collateral Ratio Monitoring

Monitor collateral ratios to prevent margin calls and liquidations:

```typescript
// Monitor collateral ratio
const monitorCollateralRatio = async (poolId: string) => {
  const { data } = await api.get(`/api/collateral/pools/${poolId}`);
  
  const ratio = parseFloat(data.data.collateralRatio);
  const marginThreshold = parseFloat(data.data.marginCallThreshold);
  const liquidationThreshold = parseFloat(data.data.liquidationThreshold);
  
  if (ratio < liquidationThreshold) {
    console.warn('⚠️ CRITICAL: Below liquidation threshold!');
  } else if (ratio < marginThreshold) {
    console.warn('⚠️ WARNING: Below margin call threshold!');
  } else {
    console.log('✓ Collateral ratio healthy');
  }
  
  return {
    ratio,
    status: ratio < liquidationThreshold ? 'critical' :
            ratio < marginThreshold ? 'warning' : 'healthy'
  };
};
```
