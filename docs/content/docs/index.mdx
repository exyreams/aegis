# Welcome to Aegis RFQ

Aegis RFQ is a confidential Request for Quote (RFQ) lending platform built on Canton Network using DAML smart contracts. It enables institutional-grade, privacy-preserving lending workflows that mirror traditional over-the-counter (OTC) debt negotiation processes.

## Key Features

- **Privacy-First Architecture**: Confidential bidding with DAML observer patterns
- **Institutional Workflows**: Multi-party contract execution with proper signatories
- **Credit System**: Comprehensive credit scoring and risk assessment
- **Collateral Management**: Automated margin calls and liquidation workflows
- **Syndicated Lending**: Multi-lender loan structures with voting mechanisms
- **Secondary Market**: Loan trading and ownership transfers
- **Yield Generation**: Liquidity pools with automated yield optimization

## Architecture

Aegis RFQ follows a three-tier architecture:

1. **Frontend** (Next.js 15 + React 19): Modern web interface with real-time updates
2. **Backend API** (Hono.js + Bun): High-performance REST API with caching
3. **DAML Ledger**: Privacy-preserving smart contracts on Canton Network

## Technology Stack

- **Smart Contracts**: DAML 2.10.2
- **Backend**: Hono.js, Bun, SQLite, Drizzle ORM
- **Frontend**: Next.js 15, React 19, shadcn/ui, Tailwind CSS 4
- **Authentication**: JWT with DAML party identification
- **Validation**: Zod schemas for type-safe APIs
