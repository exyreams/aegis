# Audit Logs API

The Audit Logs API provides comprehensive audit trails for compliance, regulatory reporting, and activity monitoring across the platform.

## Base Endpoint

```
/api/audit
```

## Platform Audit Logs

### Get Platform Audit Log

```http
GET /api/audit/platform
```

**Response:**

```json
{
  "success": true,
  "data": {
    "contractId": "00abc123...",
    "platformOperator": "AegisOperator",
    "events": [
      {
        "eventType": "PlatformFeeCollected",
        "eventDescription": "Platform fee collected from loan",
        "performedBy": "AegisOperator",
        "timestamp": "2024-06-01T12:00:00Z",
        "eventData": {
          "loanId": "LOAN-001",
          "feeAmount": "5000.00"
        }
      }
    ],
    "totalEvents": 1250
  }
}
```

### Log Platform Event

```http
POST /api/audit/platform/:contractId/log-event
```

## Code Example

```typescript
// Get loan audit trail
const getLoanAuditTrail = async (loanId: string) => {
  const { data } = await api.get(`/api/audit/loans/${loanId}`);
  
  console.log('Loan audit trail:');
  data.data.events.forEach(event => {
    console.log(`${event.timestamp}: ${event.eventType} - ${event.eventDescription}`);
  });
  
  return data;
};

// Log loan event
const logLoanEvent = async (contractId: string) => {
  const { data } = await api.post(
    `/api/audit/loans/${contractId}/log-event`,
    {
      eventType: 'PaymentMade',
      eventDescription: 'Monthly payment processed',
      performedBy: 'Alice',
      eventData: {
        paymentAmount: '50000.00',
        principalPortion: '45000.00',
        interestPortion: '5000.00'
      },
      financialImpact: {
        totalAmount: '50000.00',
        principalAmount: '45000.00',
        interestAmount: '5000.00'
      }
    }
  );
  
  return data;
};

// Monitor compliance alerts
const monitorCompliance = async () => {
  const { data } = await api.get('/api/audit/compliance-alerts');
  
  const highSeverityAlerts = data.data.filter(
    alert => alert.severity === 'High' && alert.status === 'Open'
  );
  
  if (highSeverityAlerts.length > 0) {
    console.warn(`⚠️ ${highSeverityAlerts.length} high-severity alerts require attention`);
  }
  
  return highSeverityAlerts;
};
```

## Event Types

- `LoanOriginated`
- `PaymentMade`
- `LoanDefaulted`
- `LoanRepaid`
- `LoanRestructured`

### Platform Events
- `PlatformFeeCollected`
- `LenderRegistered`
- `AssetAuthorized`
- `EmergencyShutdown`

### Compliance Events
- `ComplianceCheckPassed`
- `ComplianceCheckFailed`
- `SuspiciousActivityDetected`
- `RegulatoryReportGenerated`
