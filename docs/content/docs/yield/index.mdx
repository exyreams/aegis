# Yield Generation API

The Yield Generation API provides liquidity pool management, LP token operations, yield distribution, and automated optimization strategies.

## Base Endpoint

```
/api/yield
```

## Liquidity Pools

### List Pools

```http
GET /api/yield/pools
```

### Get Pool

```http
GET /api/yield/pools/:poolId
```

### Create Pool

```http
POST /api/yield/pools
```

**Request Body:**

```json
{
  "poolId": "POOL-001",
  "poolManager": "Bob",
  "poolName": "High Yield Lending Pool",
  "poolStrategy": {
    "strategyType": "Conservative",
    "targetAPY": "0.12",
    "riskLevel": "Low"
  }
}
```

### Join Pool

Provide liquidity to a pool.

```http
POST /api/yield/pools/:contractId/join
```

**Request Body:**

```json
{
  "participant": "Alice",
  "contributionAmount": "100000.00"
}
```

**Response:**

```json
{
  "success": true,
  "message": "Joined pool successfully",
  "data": {
    "lpTokensIssued": "100000.00"
  }
}
```

### Withdraw from Pool

```http
POST /api/yield/pools/:contractId/withdraw
```

**Request Body:**

```json
{
  "participant": "Alice",
  "lpTokensToRedeem": "50000.00"

```

```http
POST /api/yield/pools/:contractId/reinvest
```

### Distribute Yield

Pool manager distributes yield to participants.

```http
POST /api/yield/pools/:contractId/distribute-yield
```

**Request Body:**

```json
{
  "yieldAmount": "10000.00",
  "source": "Loan interest payments"
}
```

```http
POST /api/yield/pools/:contractId/deploy
```

**Request Body:**

```json
{
  "loanId": "LOAN-001",
  "amount": "500000.00"
}
```

## Code Example

```typescript
// Join liquidity pool
const joinPool = async (contractId: string) => {
  const { data } = await api.post(
    `/api/yield/pools/${contractId}/join`,
    {
      participant: 'Alice',
      contributionAmount: '100000.00'
    }
  );
  
  console.log('LP tokens issued:', data.data.lpTokensIssued);
  return data;
};

// Claim yield
const claimYield = async (contractId: string) => {
  const { data } = await api.post(
    `/api/yield/pools/${contractId}/claim-yield`,
    {
      participant: 'Alice'
    }
  );
  
  return data;
};

// Monitor pool performance
const getPoolPerformance = async (poolId: string) => {
  const { data } = await api.get(`/api/yield/pools/${poolId}`);
  
  const metrics = data.data.performanceMetrics;
  console.log('Current APY:', metrics.currentAPY);
  console.log('Total Return:', metrics.totalReturn);
  console.log('Utilization Rate:', metrics.utilizationRate);
  
  return metrics;
};
```
