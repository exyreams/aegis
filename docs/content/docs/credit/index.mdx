# Credit System API

The Credit System API provides comprehensive credit scoring, risk assessment, and credit inquiry management for borrowers and lenders.

## Base Endpoint

```
/api/credit
```

## Credit Profiles

### List Credit Profiles

Get all credit profiles visible to the authenticated party.

```http
GET /api/credit/profiles
```

**Response:**

{
  "success": true,

### Create Credit Profile

Create a new credit profile for a party.

```http
POST /api/credit/profiles
```

**Request Body:**

```json
{
  "party": "Alice",
  "profileId": "CREDIT-001",
  "initialCreditScore": 700,
  "privacyLevel": "Public"
}
```

**Response:**

```json
{
  "success": true,
  "data": {
    "contractId": "00abc123...",
    "profileId": "CREDIT-001",
    "creditScore": 700
  }
}
```

### Update Privacy Level

Change the privacy level of a credit profile.

```http
POST /api/credit/profiles/:contractId/privacy
```

**Parameters:**
- `contractId` (path): The credit profile contract ID

**Request Body:**

```json
{
  "newPrivacyLevel": "Private"
}
```

**Privacy Levels:**
- `Public`: Visible to all parties
- `Restricted`: Visible only to approved parties
- `Private`: Visible only to the profile owner

**Response:**

```json
{
  "success": true,
  "message": "Privacy level updated"
}
```

## Risk Assessments

### List Risk Assessments

Get all risk assessments for the authenticated party.

```http
GET /api/credit/risk-assessments
```

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "contractId": "00def456...",
      "assessmentId": "RISK-001",

      "assessor": "Bob",

      "riskCategory": "Medium",
      "assessmentDate": "2024-01-15T00:00:00Z",
      "status": "Approved"
    }
  ]
}
```

### Create Risk Assessment

Create a new risk assessment for a borrower.

```http
POST /api/credit/risk-assessments
```

**Request Body:**

```json
{
  "assessmentId": "RISK-001",
  "borrower": "Alice",
  "assessor": "Bob",
  "loanAmount": "1000000.00",
  "collateralValue": "1200000.00",
  "creditScore": 750,
  "debtToIncomeRatio": "0.35",
  "industryRisk": "Low",
  "marketConditions": "Stable"
}
```

**Response:**

```json
{
  "success": true,
  "data": {
    "contractId": "00def456...",
    "assessmentId": "RISK-001",
    "riskScore": 75,
    "riskCategory": "Medium",
    "recommendation": "Approve with standard terms"
  }
}
```

**Risk Categories:**
- `Low` (80-100): Minimal risk, favorable terms
- `Medium` (60-79): Moderate risk, standard terms
- `High` (40-59): Elevated risk, stricter terms
- `Very High` (0-39): Significant risk, may decline

### Python

```python
import requests

class CreditClient:
    def __init__(self, token):
        self.base_url = 'https://api.aegis-rfq.com'
        self.headers = {
            'Authorization': f'Bearer {token}',
            'Content-Type': 'application/json'
        }
    
    def create_credit_profile(self, profile_data):
        response = requests.post(
            f'{self.base_url}/api/credit/profiles',
            json=profile_data,
            headers=self.headers
        )
        return response.json()
    
    def get_credit_profile(self, profile_id):
        response = requests.get(
            f'{self.base_url}/api/credit/profiles/{profile_id}',
            headers=self.headers
        )
        return response.json()
    
    def create_risk_assessment(self, assessment_data):
        response = requests.post(
            f'{self.base_url}/api/credit/risk-assessments',
            json=assessment_data,
            headers=self.headers
        )
        return response.json()

# Usage
client = CreditClient(token='your-jwt-token')

profile = client.create_credit_profile({
    'party': 'Alice',
    'profileId': 'CREDIT-001',
    'initialCreditScore': 700,
    'privacyLevel': 'Public'
})

print('Credit profile:', profile)
```

## Credit Score Calculation

The credit score is calculated based on multiple factors:

```
Credit Score = Base Score + Payment History + Loan Performance - Defaults

Where:
- Base Score: 300-850 range
- Payment History: +50 for 100% on-time payments
- Loan Performance: +10 per successfully repaid loan
- Defaults: -100 per default
```

**Score Ranges:**
- 800-850: Excellent
- 740-799: Very Good
- 670-739: Good
- 580-669: Fair
- 300-579: Poor
